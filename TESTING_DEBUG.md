## 🐛 Типичные проблемы и решения

### Проблема: "Объявления не загружаются"

**Решение:**
```
1. Откройте Console (F12)
2. Проверьте наличие ошибок Firebase
3. Убедитесь что firebase.js содержит правильный config
4. Проверьте что коллекция "ads" существует в Firestore
```

### Проблема: "Ошибка при добавлении объявления"

**Решение:**
```
1. Проверьте что все поля формы заполнены
2. Цена должна быть положительное число
3. Проверьте Firestore Security Rules
4. Убедитесь что коллекция создана
```

### Проблема: "Поиск не работает"

**Решение:**
```
// В AdFilter.jsx ищет в реальном времени:
ad.title.toLowerCase().includes(searchValue.toLowerCase())

Убедитесь что:
1. Название объявления содержит буквы
2. Поле title существует в объявлении
3. Поиск чувствителен к буквам (A ≠ а)
```

### Проблема: "Сортировка неправильная"

**Решение:**
```
// В App.jsx сортирует по цене:
if (sortOrder === 'asc') {
  return a.price - b.price;  // Низкая → высокая
} else {
  return b.price - a.price;  // Высокая → низкая
}

Убедитесь что price - число, не строка!
```

## 🔍 Отладочные команды в консоли

### Проверка Firebase подключения
```javascript
// В консоли браузера:
import { db } from './config/firebase.js';
console.log(db); // Должен вывести Firestore instance
```

### Чтение всех объявлений
```javascript
import { db } from './config/firebase.js';
import { getDocs, collection } from 'firebase/firestore';

getDocs(collection(db, 'ads')).then(snapshot => {
  snapshot.docs.forEach(doc => {
    console.log(doc.id, '=>', doc.data());
  });
});
```

### Удаление объявления
```javascript
import { db } from './config/firebase.js';
import { deleteDoc, doc } from 'firebase/firestore';

deleteDoc(doc(db, 'ads', 'DOC_ID')); // Замените DOC_ID на реальный ID
```

## 🔐 Security Testing

⚠️ **Важно для Production:**

Текущие правила (Test Mode):
```javascript
allow read, write: if true;  // ❌ Опасно!
```

Secure правила:
```javascript
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /ads/{document=**} {
      // Читать могут все
      allow read: if true;
      // Создавать может только авторизованный пользователь
      allow create: if request.auth != null;
      // Удалять может только создатель
      allow delete: if request.auth.uid == resource.data.userId;
    }
  }
}
```
